## Challenge
Determine an expression for the complex Fourier coefficients for the following square wave:

`$$
   f(t) =
   \begin{cases}
       5 & \text{for } 0 < t < \frac{1}{2} \\
       0 & \text{for } \frac{1}{2} < t < 1
   \end{cases} 
$$`
